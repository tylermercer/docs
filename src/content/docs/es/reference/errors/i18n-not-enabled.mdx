> **i18nNotEnabled**: El módulo `astro:i18n` no se puede utilizar sin habilitar i18n en la configuración de Astro.

## ¿Qué salió mal?
El módulo `astro:i18n` no se puede utilizar sin habilitar i18n en la configuración de Astro. Para habilitar i18n, agregue un idioma predeterminado y una lista de idiomas admitidos a su configuración de Astro:
```js
import { defineConfig } from 'astro'
export default defineConfig({
 i18n: {
	 defaultLocale: 'en',
	 locales: ['en', 'fr'],
	},
})
```

Para mayor información sobre el soporte de internacionalización en Astro, consulta nuestra [Guía de Internacionalización](/es/guides/internationalization/).
