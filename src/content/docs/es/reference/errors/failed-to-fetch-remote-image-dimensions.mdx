> No se han podido obtener las dimensiones de `IMAGE_URL`.

## ¿Qué salió mal?

Error al determinar las dimensiones de la imagen remota. La causa suele ser una URL incorrecta o el intento de inferir el tamaño de una imagen de la carpeta pública, lo cual no es posible.
